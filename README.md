## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/To0424/AI-career.git
cd AI-career
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5174`

## Available Scripts
